# Cadence AI - Intelligent Scheduling Assistant

AI-powered scheduling and productivity assistant. Helps manage time, plan tasks, and optimize daily workflows.

**Stack:** Next.js, Supabase, TypeScript, Tailwind CSS

## Features

- **AI Scheduling** - Intelligent time block suggestions
- **Task Management** - Priority-based task organization
- **Calendar Integration** - Visual schedule management
- **Productivity Insights** - Track and improve daily output
